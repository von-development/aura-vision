## Aura Vision

Develop a lightweight computer-vision module for real-time object detection, classification, and distance/size estimation.


# Scope 

The goal is a real-time interface returning bounding boxes, class labels, distance, and size.

The final outcome is to have a working demo working in a laptop webcam. 

Addiotnaly, try to integrate it with the rasberry pi for the Aura Project being develop in the adcanced topics of robotic class.
